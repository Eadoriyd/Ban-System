# Ban-System
Приложение для игры Эадор : Новые Горизонты, позволяющее блокировать здания в замке и заключение союза с расами (пока не реализовано).

Установка: Распаковать архив в папку с Эадором, что б EadorBanSystem.exe был на уровне с Eador.exe.

Как пользоваться: Запустить EadorBanSystem.exe. В левой части программы можно выбрать какие баны будут применены на игру простым кликом на картинку. Повторный клик убирает бан. Правая часть содержит список сохранений и кнопки. Список сохранений изначально имеет Default - он всегда пустой, его нельзя удалить, переместить или перезаписать. Кнопки делают следующее: 
Play: просто запускает Eador.exe из этой же папки. 
Apply+Save: Применяет выбранные баны и перезаписывает текущее сохранение. Кнопка жёлтая, если баны в игре не соответствуют тем, что нарисованы в левой панели. Зелёная - если соответствуют. Если баны не пустые - нельзя сохранить в Default, будет предложено создать новое сохранение. 
Save: Перезаписывает текущее сохранение. Кнопка жёлтая, если баны в выбранном сохранении из списка не соответствуют тем, что нарисованы в левой панели. Зелёная - если соответствуют.Если баны не пустые - нельзя сохранить в Default, будет предложено создать новое сохранение. 
Clear: Очистить баны. 
Load: Загрузить выбранное сохранение. 
New Save: Создать новое сохранение с текущими банами из левой панели. 
Delete: Удалить текущее сохранение (но не Default). 
Кнопки вниз/вверх: Позволяют двигать сохранения вверх/вниз отностиельно остальных.

Остальное взаимодействие: 
Стрелки на клавиатуре вверх/вниз (влево/вправо): Навигация по списку сохранений. 
Горячая клавиша Delete: Удалить текущее сохранение. 
Дабл-клик по сохранению: Загрузить выбранное сохранение.
